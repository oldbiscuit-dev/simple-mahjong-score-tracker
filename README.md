# Simple Mahjong Score Tracker

Mahjong Score Tracker is a simple iOS ledger for tracking Mahjong scores, funds, seating, results, and exports.

## Current workflow

1. First install shows the Welcome screen.
2. Start New Game opens the setup flow and keeps Archives intact.
3. User enters four player names, starting funds, and setup rules such as Winner Repeat.
4. Start Game automatically shuffles seats and opens the Record page.
5. Record page is used during play.
6. Players page shows seats, funds, and low-funds warnings.
7. Results page shows leaderboard, history, archives, and settlement information.
8. Settings contains scoring, currency, export, help, and reset tools.
